### vuepress-theme-hope 是一个轻量主题

## 使用主题

借助 Vuepress 强大的主题系统，启用主题从未如此简单。

### 安装

```bash
npm i -D vuepress-theme-hope
```

### 使用

```js
// .vuepress/config.js
module.exports = {
  theme: 'hope'
};
```

### 完全兼容

引入主题后，无需进行任何配置，主题即可正常工作！
