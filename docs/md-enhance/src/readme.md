### Install

```bash
npm i -D vuepress-plugin-md-enhance
```

Or

```bash
yarn add -D vuepress-plugin-md-enhance
```

### Usage

```js {3-7}
// .vuepress/config.js
module.exports = {
  plugin: [
    "md-enhance",
    {
      // your options
    },
  ],
};
```
